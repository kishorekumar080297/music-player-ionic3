# Music Player Android app

Music player android app developed using Ionic 3.

## Prerequisites ##

Install Node.js

Install Cordova & Ionic,

```
$ npm install -g cordova ionic
```

Install npm packages

```
$ npm install
```

Run through Browser

```
$ ionic serve
```

To run on a device, connect the device and enable adb

```
$ ionic run android
```
